# GOWHERE - Khám Phá Thiên Đường Ẩm Thực Của Bạn

![GOWHERE Banner](https://images.unsplash.com/photo-1504674900247-0877df9cc836?auto=format&fit=crop&q=80&w=2070)

### Review thật

**GOWHERE** không chỉ là một trang web tìm kiếm đồ ăn, mà là người bạn đồng hành tin cậy trên hành trình chinh phục vị giác của bạn. Chúng tôi mang đến một hệ sinh thái kết nối những tâm hồn ăn uống, nơi mỗi địa điểm đều được xác thực và mỗi đánh giá đều mang giá trị trải nghiệm thực tế.

---

## Điểm Nổi Bật

**Tìm Kiếm Thông Minh:** Hệ thống lọc theo Tỉnh thành/Quận huyện siêu mượt mà. Bạn đang ở Tiền Giang? Chỉ cần 1 click để thấy mọi tinh hoa ẩm thực Mỹ Thọ, Cai Lậy...

**Giao Diện Premium:** Trải nghiệm người dùng đỉnh cao với phong cách thiết kế hiện đại, hiệu ứng Glassmorphism và tối ưu hóa hoàn hảo cho cả Mobile & Desktop.

**Dữ Liệu Sống:** Hệ thống quản lý hành chính thông minh giúp cập nhật Real-time số lượng quán đang hoạt động tại từng khu vực.

**Cộng Đồng Góp Ý:** Hệ thống Tag tùy chỉnh giúp bạn phân loại quán theo đúng "gu": *#QuánVỉaHè, #ViewĐẹp, #GiáSinhViên, #HẹnHò...*

## Cài Đặt Khởi Chạy

Nếu bạn muốn trải nghiệm bản local trên máy tính của mình:

1. **Clone repository:**
   ```bash
   git clone https://github.com/yourusername/foodiemap.git
   ```

2. **Cài đặt dependencies:**
   ```bash
   npm install
   ```

3. **Cấu hình môi trường:**
   Tạo file `.env` và thêm:
   ```env
   MONGODB_URI=your_mongodb_connection_string
   JWT_SECRET=your_secret_key
   ```

4. **Khởi chạy:**
   ```bash
   npm run dev
   ```
